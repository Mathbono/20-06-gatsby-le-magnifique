# Gatsby le Magnifique

## Blog de pages Markdown générées par CMS intégré

Gatsby fait du dynamique de la génération de statique. Via [https://20-06-gatsby-le-magnifique.netlify.app/admin/](https://20-06-gatsby-le-magnifique.netlify.app/admin/), on peut comme sur Wikipédia générer les pages du blog.

## Fonctionnalités utilisateur

-  Accès au blog dynamiquement page par page comme dans un livre
-  Possibilité d’envoyer son email pour être notifié
